## Key Decisions
- ElevenLabs is proxied through EVOKORE-MCP (not user-scoped) so all connected clients get voice tools.
- VoiceMode is user-scoped (needs direct microphone/speaker access, can't be proxied).
- Output mode set to `both` (files + base64) for maximum flexibility.

## Technical Notes
- `cross-spawn` (used by MCP SDK) resolves `uvx.exe` on Windows PATH without needing `.cmd` suffix.
- ElevenLabs `elevenlabs-mcp` package validates `ELEVENLABS_API_KEY` at import time (line 60 of server.py), so the env var must be present before the child process starts.
- VoiceMode `VOICEMODE_TTS_SPEED` supports up to 4.0x when using OpenAI TTS provider.
